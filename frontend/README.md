### Frontend Project Structure
```
src/
├── assets/         # 정적 자원 (이미지, 로고, 효과음 등)
├── components/     # 재사용 가능한 UI 컴포넌트 (버튼, 입력창, HUD 등)
├── pages/          # 화면 단위 페이지 (Setup, Simulation, Report)
├── services/       # API 통신 로직 (Claude AI API 연동)
├── styles/         # 전역 및 컴포넌트 스타일 (CSS)
├── App.jsx         # 메인 라우터 및 전역 상태 관리
└── main.jsx        # 애플리케이션 엔트리 포인트
```
